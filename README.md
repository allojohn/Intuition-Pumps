## 

用詹姆斯·马奇的模型、故事、行动框架来看（参考：[经验的疆界 -豆瓣](https://book.douban.com/subject/27146611/)），intuition pumps 侧重**从故事角度**描述。读者们除了搞懂故事的含义之外，更应该做的，其实是学会尝试复述丹尼特的这些故事与叙述方式，讲给后来者。
