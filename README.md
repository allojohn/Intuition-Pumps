## 尴尬的发现

在已经写了一大半的时候，我突然发现，写intuition pumps 笔记本身，恰恰违背了丹尼特的本义。用马奇的模型、故事、行动框架来看，intuition pumps 侧重从故事角度描述。我应该做的，是去尝试复述丹尼特的这些故事，而不是琢磨怎么把故事的道理给推演出来。
